# AIESEC in MUST Podio

Full-stack Operations Management Platform built for **AIESEC in MUST Local Committee** to centralize and streamline all core functions into one secure, scalable system.

---

## Overview

A production-ready internal platform serving as the LC’s **Single Source of Truth**, replacing manual tracking and scattered spreadsheets with a structured API-driven architecture.

## Architecture

### Backend
- Node.js
- Express REST API
- SQLite (via better-sqlite3) for persistent storage
- JWT-based Authentication
- Multer middleware for file uploads

### Frontend
- HTML5 / CSS3 / Vanilla JavaScript
- Fully API-integrated (no in-memory state)
- Responsive Dashboard UI

---

## Core Modules

- Members Management
- OGX (Outgoing Exchange)
- ICX (Incoming Exchange)
- B2B & Partnerships Pipeline
- Activity Logs & Internal Tracking
- MUST SU Proxy Integration

## Key Features

- Secure JWT Authentication

## How to Run Locally 🏃‍♂️

1. Unzip the `Podio-FullStack.zip` file.
2. Open your terminal inside the extracted project folder.
3. Run the command (npm start) or (node server.js).
4. Visit http://localhost:3000 in your web browser.
